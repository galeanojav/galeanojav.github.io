# Personal academic website (GitHub Pages)

This repository hosts a simple academic website built with **Jekyll** and the GitHub Pages-supported theme **Cayman**.

## Quick publish
1. Create a new GitHub repository (e.g., `javier-galeano` or `galeanojav.github.io`).
2. Upload all files in this repo.
3. In GitHub: **Settings â†’ Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` / folder `/ (root)`
4. Wait for the Pages URL to appear.

## Editing content
- Update the pages: `index.md`, `publications.md`, `projects.md`, `talks.md`, `software.md`, `teaching.md`, `cv.md`, `contact.md`
- Add PDFs (papers, slides) into `assets/` and link them.
